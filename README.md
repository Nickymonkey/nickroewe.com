# Jekyll Site

A simple static website built with Jekyll and hosted for free on GitHub Pages at [nickroewe.com](https://nickroewe.com).

## Architecture Overview

### Technology Stack

**Static Site Generator:**
- Jekyll (Ruby-based)
- Markdown for content
- Liquid templating engine

**Hosting:**
- GitHub Pages (free hosting)
- Automatic HTTPS
- Custom domain support

**Version Control:**
- GitHub repository
- Git-based workflow

### System Architecture

```
┌─────────────────┐
│   User Browser  │
└────────┬────────┘
         │ HTTPS
         ▼
┌─────────────────┐
│  GitHub Pages   │
│   (CDN)         │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  Static HTML    │
│  (Generated by  │
│   Jekyll)       │
└─────────────────┘
```

### Project Structure

```
nickroewe.com/
├── _config.yml          # Jekyll configuration
├── _layouts/            # HTML layouts
├── _includes/           # Reusable components
├── _sass/               # Sass stylesheets
├── assets/              # CSS, JS, images
├── index.html           # Homepage
├── projects.md          # Projects page
├── experience.md        # Experience page
├── Gemfile              # Ruby dependencies
└── README.md
```

### Deployment Strategy

1. **Development Environment**
   - Local Jekyll server (`bundle exec jekyll serve`)
   - Live reload for development
   - Preview at `http://localhost:4000`

2. **Production Environment (GitHub Pages)**
   - Automatic deployment on push to main branch
   - Jekyll builds site automatically
   - Live at `https://nickroewe.com` (custom domain)
   - Also available at `https://[username].github.io/nickroewe.com/`

### Key Features

- Simple, fast static site
